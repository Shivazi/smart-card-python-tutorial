# Middleware to access a smart card using PC/SC (daemon side) 
https://www.howtoinstall.co/en/ubuntu/xenial/pcscd

Install pcscd:
```
sudo apt-get update
sudo apt-get install pcscd

show: lsusb

sudo apt install pcsc-tools
pcsc_scan
```
